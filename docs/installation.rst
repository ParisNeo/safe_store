============
Installation
============

Install ``safe_store`` using pip. Starting from version 3.4.2, ``safe_store`` follows a "batteries-included" approach, meaning all dependencies for parsing, vectorization (ST, OpenAI, Ollama, Cohere, TF-IDF), and encryption are installed by default.

.. code-block:: bash

   pip install safe_store

Development Dependencies
------------------------

To install dependencies needed for testing, building, and documentation generation:

.. code-block:: bash

   pip install "safe_store[dev]"
